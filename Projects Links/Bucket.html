Programming Language: Bucket
Created by Mateus Morais. Copyright (c) 2019 by Mateus Morais.

Bucket means Basic Unity-Control to Kindle Eletrical Tapes, a language to create sand castles and input-output codes. Bucket can get, read, change and print the user text as way you want. Download the project in the Github.

A hello world in Bucket(helloworld.bk): 


                [to Basic]

                #helloworld in Bucket:
                
                    bucket open:
                
                        show 'Hello World!'
                
                    end m.
                close.
